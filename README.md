# Enigmatry frontend task

## Quick start guide

1. Clone or download this repo onto your computer.
2. Install [Node.js](https://nodejs.org/en/) if you don't have it.
3. Open up this file with your favorite code editor
4. Open terminal and run

```bash
npm install
```

- Next run this command in your terminal to start [parcel](https://parceljs.org/) dev server

```bash
npm start
```

Check out the final project [here]("https://www.google.com/)
